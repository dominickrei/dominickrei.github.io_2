8190 Final Project - Image Similarity Using Deep Neural Networks

Dec 7, 2022 1 min read

Description

This is a cloud-based image search system. An image dataset of images is hosted in an AWS S3 bucket in the cloud that can be queried using any given image. Querying corresponds to finding the image in the S3 bucket that is most similar to the query image. The similarity measure used is cosine similarity between the neural network representations.

Dataset

The dataset I used is ImageNet, a large-scale image dataset of 1.3 million images of various resolutions. Each image is associated with a single class and there are 1000 classes in total

Tools used

  • Deep learning
    • Libraries: TensorFlow, Keras, numpy, Pillow
    • Network Architectures: ResNet50 pretrained on ImageNet
  • Cloud Computing
    • Libraries: PyArrow, pandas
    • Frameworks: Spark
